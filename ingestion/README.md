# Ingestion Scripts

Scripts Python (Google Colab) utilizados para:
- Baixar dados do Kaggle
- Processar e validar dados
- Carregar no AWS RDS

## Arquivos

- `kaggle_download.py` - Script para download dos dados do Kaggle
- `rds_loader.py` - Script para carregamento dos dados no AWS RDS

## Como Usar

1. Abra os notebooks no Google Colab
2. Configure as credenciais do Kaggle e AWS
3. Execute as células conforme instruído
